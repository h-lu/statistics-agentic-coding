# Week 02 评分标准

## 基础练习（60 分）

### 练习 1：集中趋势分析（20 分）

| 评分点 | 分值 | 说明 |
|--------|------|------|
| 分组计算正确 | 5 分 | 使用 `groupby` 或 `pivot_table` 按物种分组 |
| 均值和中位数计算准确 | 5 分 | 数值正确，保留合理小数位 |
| 样本量统计正确 | 3 分 | 包含 count 列，数值准确 |
| 分析清晰 | 7 分 | 能解释"均值和中位数的差异说明什么"，能根据数据特点选择合适的统计量 |

**加分项**：
- 能指出哪个物种的分布最对称/最偏态（均值和中位数最接近/最远）：加 1 分

**扣分项**：
- 只看均值就下结论，没考虑中位数：扣 3 分
- 忘记标注样本量：扣 2 分
- 分析时只说"Gentoo 最大"，没解释"为什么"：扣 2 分

---

### 练习 2：离散程度分析（20 分）

| 评分点 | 分值 | 说明 |
|--------|------|------|
| 标准差计算正确 | 5 分 | 使用 `std()` 方法，数值准确 |
| IQR 计算正确 | 5 分 | Q75 - Q25，公式正确 |
| 极差计算正确 | 3 分 | max - min，数值准确 |
| 分析清晰 | 7 分 | 能解释"波动"的实际意义，能对比不同物种之间的波动差异 |

**加分项**：
- 能解释"为什么标准差和 IQR 数值不一样"：加 1 分

**扣分项**：
- IQR 计算错误（例如用 Q75/Q25 而不是差值）：扣 3 分
- 只给数字不解释，没有说明"波动大意味着什么"：扣 3 分
- 忘记比较不同物种之间的波动差异：扣 2 分

---

### 练习 3：分布可视化（20 分）

| 评分点 | 分值 | 说明 |
|--------|------|------|
| 直方图质量 | 8 分 | 按物种分组，使用半透明颜色，bins 数量合适 |
| 箱线图质量 | 7 分 | 按物种分组，能清楚看到异常值 |
| 图表完整性 | 3 分 | 有完整的标题和标签 |
| 分析清晰 | 2 分 | 能结合图表内容，提到具体的观察结果 |

**扣分项**：
- 直方图没有使用半透明颜色：扣 2 分
- 图表缺少标题或标签：扣 1-2 分
- 分析太泛（例如"分布看起来不错"）：扣 1 分

## 进阶练习（30 分）

### 练习 4：一页分布报告（20 分）

| 评分点 | 分值 | 说明 |
|--------|------|------|
| 报告结构完整 | 5 分 | 包含数据概览、描述统计表、关键发现、图表 |
| 描述统计表准确 | 4 分 | 数据准确，格式清晰，包含所有必需列 |
| 关键发现有价值 | 7 分 | 具体有洞察，不是泛泛而谈，至少 3 条 |
| 代码能自动生成 | 4 分 | Python 脚本能自动生成报告（图表 + Markdown） |

**加分项**：
- 关键发现特别深入，提到"双峰分布"等细节：加 1-2 分

**扣分项**：
- 关键发现太泛（例如"数据很有趣"）：扣 3 分
- 报告和图表分离，没有整合：扣 2 分
- 代码只 print 不写文件：扣 2 分

## 通用评分标准（适用于所有练习）

### 代码质量（额外扣分项）

| 问题 | 扣分 |
|------|------|
| 代码无法运行 | 该题 0 分 |
| 变量命名混乱（如 `a`、`b`、`x`、`tmp`） | 扣 1-2 分 |
| 完全没有注释 | 扣 1 分 |
| 硬编码绝对路径 | 扣 2 分 |
| 大量复制粘贴代码（未封装函数） | 扣 2 分 |

### 可复现性（额外扣分项）

| 问题 | 扣分 |
|------|------|
| 图表只 `plt.show()` 不保存文件 | 扣 1 分 |
| 未说明数据来源 | 扣 1 分 |
| 随机操作未设置种子 | 扣 1 分 |
| 没有 README 或运行说明 | 扣 1 分 |

### 诚实可视化（额外扣分项）

| 问题 | 扣分 |
|------|------|
| 柱状图 Y 轴未从 0 开始且未说明 | 扣 2 分 |
| 没有标注实际数值 | 扣 1 分 |
| 没有标注样本量 | 扣 1 分 |

## 反馈建议模板

对学生作业的反馈可以参考以下格式：

```markdown
## Week 02 作业反馈

**总分**：XX / 100

### 做得好的地方
- 练习 1：能准确计算均值和中位数，分析部分提到了"Gentoo 的均值和中位数很接近，说明分布比较对称"。
- 练习 3：直方图使用了半透明颜色，分组重叠部分清晰可见。
- 练习 4：报告结构完整，关键发现提到了"双峰分布"等细节。

### 需要改进的地方
- 练习 1：分析部分只说"Gentoo 最大"，没解释"均值和中位数的差异说明什么"。下次可以加一句："均值和中位数接近说明分布比较对称，没有极端值拉偏均值。"
- 练习 2：只给了数字，没有解释"波动大意味着什么"。建议加一句："Gentoo 的体重波动最大，说明这个物种内部差异更大，可能受年龄、性别等因素影响。"
- 练习 3：图表缺少标题。建议加上 `plt.title("Penguin Body Mass Distribution by Species")`。

### 下周建议
- 继续关注"选择合适的统计量"这个话题：什么时候用均值，什么时候用中位数
- 尝试在报告里加入更多"数据的故事"：不只是数字，还要解释"这意味着什么"
```
