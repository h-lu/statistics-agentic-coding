# 数据卡（Data Card）

## 数据来源

- **数据集名称**：Palmer Penguins
- **来源**：seaborn 内置数据集
- **原始来源**：Palmer Station, Antarctica LTER
- **描述**：南极 Palmer Station 的三种企鹅（Adelie, Chinstrap, Gentoo）的形态测量数据
- **收集时间**：2007-2009 年
- **单位说明**：长度单位为毫米（mm），重量单位为克（g）
